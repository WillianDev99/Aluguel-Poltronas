# ComfortCare - Gestão de Poltronas Pós-Cirúrgicas

Este projeto contém tudo o que você precisa para rodar o aplicativo de gestão de locações de poltronas pós-cirúrgicas localmente.

## Como rodar localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`
2. Configure as variáveis de ambiente necessárias no arquivo `.env`.
3. Inicie o aplicativo:
   `npm run dev`
